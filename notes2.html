Nmap — Port Scanning Nmap

Basic host and port discovery commands.

# Full TCP scan with service/version detection nmap -sC -sV -p- <target>
# Fast scan — top 1000 ports nmap -T4 -F <target>
# UDP scan nmap -sU --top-ports 100 <target>
# OS detection + aggressive nmap -A -O <target>
